Usar **1A** si hay datos; si no, **YTD**. Anualizar cuando corresponda. * **Tracking Error (TE)** = Desv. estándar de (rₜ ETF − rₜ Índice) con **datos diarios**, ventana **1A**. Si < 250 días: **“N/D (historia insuficiente)”**. * **TCO práctico** = TER + spread medio (bps) + prima/descuento esperada + fricción (si RF: riesgo de creación/redención en estrés). * **Liquidez operable (umbral)**: **ADV 20d ≥ USD 5–10M** y **spread ≤ 20 bps**; si no, usar **órdenes limitadas** y **entradas escalonadas**. * **Renta fija**: prioriza **SEC Yield (30-day)**; reporta **YTM/YTW**, **Duración efectiva/modificada**, **OAS** si hay. * **Commodities**: estructura (contango/backwardation), **calendario de rolado**, **roll yield**, colateral, **estructura legal** (trust/ETF/ETN). * **Domicilio/Régimen**: US/IE/LU; UCITS/’40 Act; **WHT** estimado; **FX hedged** (coste/beneficio por diferencial de tasas). * **SL base**: **−10%** fijo. **Opción ATR** solo si volatilidad > percentil 80 del último año (entonces ATR manda). * **RRR objetivo ≥ 2.0**; **Riesgo% del capital ≤ 2%**. --- # **Informe de Inversión (ETFs)** ## 1) **Encabezado** * **Fecha de análisis**: `[YYYY-MM-DD]` (America/Bogota) * **Ticker**: `[TICKER]` – `[Nombre completo]` * **Índice/Objetivo**: `[Nombre del índice o mandato activo]` * **Capital total**: `[capital]` * **Inversión asignada**: `[inversion]` * **Clasificación operativa**: `Configuración favorable / Esperar confirmación / No favorable` --- ## 2) **Datos Generales del ETF** **Tabla** (Elemento | Detalle | **Fuente** | **Fecha**) | Elemento | Detalle | Fuente | Fecha | | ----------------------- | --------------------------------------------------- | ------ | ----- | | Precio actual (USD) | `[USD …]` | | | | NAV por participación | `[USD …]` | | | | Prima/Descuento vs NAV | `[± … %]` | | | | Clase de activo | `[Equity / Renta fija / Commodities / Multi-asset]` | | | | Región / Exposición | `[USA / Global / EM / Europa / Sector / Tema]` | | | | Índice / Metodología | `[Cap-weighted / Equal-weight / Factor / Activo]` | | | | Emisor | `[iShares / Vanguard / State Street / …]` | | | | **TER** (gasto anual) | `[… %]` | | | | **AUM** | `[USD …]` | | | | Inception | `[YYYY-MM-DD]` | | | | Réplica | `[Física completa / Sampling / Sintética (swap)]` | | | | **Domicilio & Régimen** | `[US/IE/LU; evitar slippage en apertura/cierre. * **Tracking**: TD/TE anómalos → investigar. * **FX**: si no-hedged, documentar sensibilidad al USD. * **Renta fija**: duración/convexidad elevadas → riesgo de tasas. * **Commodities**: contango severo → riesgo de roll negativo. * **Estructurales**: sintéticos (swap) → riesgo contraparte; política de préstamo de valores. **Semáforos:** * ⛔ **Riesgo% > 2** → *No favorable* * ⛔ **RRR < 1.5** → *Riesgo excede recompensa* * ⚪ **1.5 ≤ RRR < 2.0** → *Precaución* * ✅ **RRR ≥ 2.0** → *Aprobación* **Gestión operativa personal (marco):** * Enfoque en **controlar riesgo**, no en predecir timing. * **SL fijo −10%** (base) para evitar salidas por ruido; **ATR** solo si vol alto (p80). * Horizonte: **swing o largo plazo (3–12 meses)**. * **Entradas escalonadas** permitidas en activos de alta calidad. * Criterio de salida: romper −10% o alcanzar niveles técnicos/fundamentales de reevaluación. * Prioriza **riesgo monetario ≤ 2% del capital** sobre precisión del punto de entrada. * El modelo puede **permitir compras parciales/acumulación** aun si la señal técnica no es ideal (si la calidad es alta). --- ## 7) **Comparativa de ETFs Pares** | ETF (Ticker) | TER | AUM | TD 1A | TE 1A | YTD | 1A | 3A | Spread | ADV | Fuente | Fecha | | ------------------- | --: | --: | ----: | ----: | --: | -: | -: | -----: | --: | ------ | ----- | | `[ETF actual]` | | | | | | | | | | | | | `[Peer 1]` | | | | | | | | | | | | | `[Peer 2]` | | | | | | | | | | | | | `[Alternativa idx]` | | | | | | | | | | | | **Conclusión comparativa:** `[Coste vs eficiencia vs liquidez;
